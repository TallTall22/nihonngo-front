# 專案初始化
## **前置作業**
已安裝 node 和 npm

<br>

## **安裝**
1. Clone 專案
```
 git clone https://github.com/TallTall22/nihonngo-front

 cd nihonngo-front
```

<br/>

2. 安裝套件
```
npm install
```

<br/>

3.  連接後端
請詳細閱讀後端 Readme [Back-end](https://github.com/TallTall22/nihonngo-back) 
```
git clone https://github.com/TallTall22/nihonngo-back
cd nihonngo-back
npm install
```



<br/>

## **啟動專案**

1. 啟動專案
```
npm run dev
```

如果成功會看到 vite 啟動的 Local 網址

<br>

2. 停止專案
```
control + c
```
